## Setup and Launch

1. Clone repository:

```bash
   git clone https://github.com/lifeofcapo/portals-fintopio.git
```

2. Download dependecies:

```bash
   npm install
```

3. Launch dev server:

```bash
   npm run dev
```

4. For production:

```bash
npm run build
```
